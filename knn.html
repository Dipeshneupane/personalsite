Potato Leaf Disease Detectation using KNN

The Potato Leaf Disease Detection project aims to create a robust system for identifying and classifying diseases affecting potato plants based on leaf images. The project utilizes the K-Nearest Neighbors (KNN) algorithm, a machine learning technique known for its simplicity and effectiveness in classification tasks. Numpy and Pandas are employed for data manipulation and preprocessing, while Flask is used to develop a user-friendly web application for seamless interaction.

Key Features:

1. Image Datasets:
  • A diverse dataset of potato leaf images, encompassing various healthy and diseased states, is collected and curated.
  • Each image is annotated and labeled to train the KNN algorithm for accurate disease detection.
2. Preprocessing using Numpy and Pandas:
  • Numpy is utilized for image processing tasks such as resizing, normalization, and feature extraction.
  • Pandas is employed for organizing and managing the dataset, ensuring efficient handling of data during training and testing phases.
3. K-Nearest Neighbors (KNN) Algorithm:
  • The KNN algorithm is implemented to classify potato leaf images into different disease categories based on their features.
  • The model is trained on the preprocessed dataset to learn patterns and relationships between various leaf conditions.
4. Web Application using Flask:
  • Flask is employed to develop a user-friendly web application for seamless interaction with the disease detection system.
  • Users can upload images of potato leaves through the web interface, and the system will provide real-time predictions regarding the presence of diseases.
5. Results Display:
  • The web application displays the results of disease detection, providing users with insights into the health status of their potato plants.
