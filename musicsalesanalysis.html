Music Sales Analysis(SQL + Excel + Tableau)

The Music Sales Analysis project is from extracting valuable insights from music sales data using a combination of SQL for data aggregation, Excel for data cleaning and transformation, and Tableau for data visualization.

Key Objectives:

1. SQL(Data Aggregation):
  • Aggregate data from various tables to provide a unified view of music sales information.
  • Utilize SQL queries to join, filter, and aggregate data, creating a consolidated dataset for analysis.
    • 2. Excel(Data Cleaning and Transformation):
    • Address data inconsistencies, missing values, and outliers to ensure data accuracy.
    • Employ Excel functions and tools for data transformation, enabling the creation of a clean and structured dataset for further analysis.
    3. Tableau(Data Visualization):
    • Import the cleaned and transformed data into Tableau for intuitive and insightful data visualization.
    • Create interactive dashboards and reports to showcase key metrics, trends, and patterns within the music sales dataset.
    • Leverage Tableau's capabilities to provide a visually compelling representation of sales performance, customer behavior, and product popularity.
