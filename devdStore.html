DevD Store Sales Analysis(Excel)

The DevD Store Sales Analysis project create the dashboard to analyize the sample sales data using only Microsoft Excel. The objective is to gain the knowledge of excel for data analysis such as pivot table, slicer, power query etc.

Key Objectives:

1. Data Exploration:
  • Thoroughly examined sample sales data to understand its structure, identify patterns, and uncover potential trends.
    • 2. Data Cleaning:
    • Addressed any inconsistencies, missing values, or outliers within the dataset to ensure accuracy and reliability in subsequent analyses.
    3. Sales performance Metrics:
    • Utilized Microsoft Excel functions and tools to calculate key sales performance metrics such as revenue, profit margins, and growth rates.
    4. Visualization:
    • Created visually appealing and informative charts and graphs to represent sales trends, product performance, and other relevant insights.
